## Image Guidelines

- **Format:** JPEG for photos (optimized for web)
- **Color scheme:** Complement grey × forest green theme
- **Tone:** Professional, modern, calm
- **Quality:** High resolution, properly compressed
- **Lighting:** Natural, soft, professional
- **Mood:** Trustworthy, capable, approachable

## Optimization Tips

Before uploading images:
1. Resize to recommended dimensions
2. Compress using tools like TinyJPG or ImageOptim
3. Target file size: 200-500KB per image
4. Maintain quality while reducing file size
5. Use descriptive filenames (as listed above)

## Stock Image Sources

Recommended sources for professional stock images:
- Unsplash (free, high quality)
- Pexels (free, commercial use)
- Adobe Stock (premium)
- Shutterstock (premium)
